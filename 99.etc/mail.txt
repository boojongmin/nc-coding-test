과제: 시계열 데이터 이상 탐지 예측 모델을 연동한 간단한 웹 페이지 구현
파일형식: 구현 소스 및 실행 가능한 패키지와 설명문서


--------------------
1.     뷰 부분(가산점)에서 순수 dom api는 JQuery 를 사용하지 않고 구현해야 합니다.

 

2.     Http Restful API 개발(가산점) 부분에서, 
API 를 직접 구현할 때는 예측 대상이 되는 데이터를 전달하고 예측 결과를 응답 받는 구조로 구현해야 합니다. 
미리 예측 결과를 만들어 두고 단순히 그 결과를 조회하는 방식은 허용하지 않습니다.

 

3.     크로스 도메인 접근이 해제가 되어있지 않았으나, 현재 접근 허용 처리하였습니다.
혹 문제 진행 시 어려움이 있으셨다면 재 진행 부탁 드립니다.
