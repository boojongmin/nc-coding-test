# NC 코딩 테스트

## 폴더 설명
- 01.front
  - dom 관련 jquery등 라이브러리 없이 구현한 버전
- 02.front+server
  - vuejs를 이용하여 SPA로 개발한 버전
  - 프로젝트 설명
    - nc-test-front
      - vue를 사용하여 구현한 프론트사이드 프로젝트
    - nc-test-server
      - springboot2를 사용하여 구현한 서버 사이드 프로젝트

## 사용라이브러리
